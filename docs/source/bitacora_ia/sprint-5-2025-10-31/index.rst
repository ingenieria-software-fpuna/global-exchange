Resumen del Sprint
------------------

**Período**: 31 de Octubre, 2025

**Objetivos principales**:

1. Agregar gestión de denominaciones al modelo
2. Mejorar identificación de transacciones
3. Implementar filtros de monedas por método de pago
4. Mejorar visualización de cotizaciones

**Historias de usuario completadas**:

- Gestión de denominaciones de billetes y monedas
- Sistema de códigos de verificación para transacciones
- Filtros dinámicos de monedas según método de pago
- Vista mejorada de cotizaciones con datos adicionales

**Tecnologías utilizadas**:

- Django 5.x
- PostgreSQL
- Bootstrap 5
- JavaScript para interactividad

**IA utilizada**:

- Cursor AI para desarrollo de funcionalidades
- Asistencia en refactoring y optimización
- Generación de tests unitarios
